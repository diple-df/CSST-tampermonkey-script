# Custom Search System Tabs (CSST)

> Tampermonkey-скрипт для настройки панели вкладок поиска в Google, Яндекс, Bing и DuckDuckGo.  
> Скрывайте ненужные вкладки, переименовывайте их, перенаправляйте на другие сервисы и добавляйте свои кнопки прямо в панель навигации поисковика.

## Установка

### Шаг 1 — Установите менеджер пользовательских скриптов

Скрипт работает через расширение-менеджер для браузера. Поддерживаются:

| Расширение | Chrome | Firefox | Edge |
|---|---|---|---|
| **Tampermonkey** (рекомендуется) | ✓ | ✓ | ✓ |
| Violentmonkey | ✓ | ✓ | ✓ |
| Greasemonkey | — | ✓ | — |

Установите расширение из официального магазина вашего браузера.

### Шаг 2 — Установите скрипт

**Вариант А — через Raw-ссылку GitHub (самый простой):**

1. Откройте файл `csst_tmprmnk.js` на GitHub
2. Нажмите кнопку **Raw**
3. Tampermonkey автоматически распознает скрипт и предложит его установить
4. Нажмите **Install**

**Вариант Б — вручную:**

1. Скопируйте содержимое файла `csst_tmprmnk.js`
2. Откройте Tampermonkey → **Create a new script**
3. Вставьте код и нажмите **Сохранить** (`Ctrl+S`)

### Шаг 3 — Откройте любой поисковик

Перейдите на `google.com`, `yandex.ru`, `bing.com` или `duckduckgo.com`, введите любой запрос — в строке вкладок появится кнопка настроек (≡).

## Поддерживаемые поисковые системы

| Поисковая система | Домены | Акцентный цвет |
|---|---|---|
| **Google** | `google.com`, `google.co.uk`, `google.ru` и все прочие региональные домены | Синий `#4285f4` |
| **Яндекс** | `yandex.ru`, `yandex.com`, `yandex.by`, `yandex.kz`, `ya.ru` | Красный `#fc3f1d` |
| **Bing** | `bing.com` | Бирюзовый `#008373` |
| **DuckDuckGo** | `duckduckgo.com` | Оранжевый `#de5833` |

Настройки хранятся **отдельно для каждой поисковой системы** — изменения в Google не влияют на Яндекс.


## Основные функции

### Скрыть вкладку

Любую стандартную вкладку поисковика можно полностью убрать из панели. Скрипт прячет и саму ссылку, и её обёртку (`<li>`, `<div>`), чтобы не оставалось пустых мест.

Вкладки, скрытые самим поисковиком (например, «Товары» у Яндекса), скрипт не трогает — открываются только те, которые скрыл именно он.

### Переименовать вкладку

Вы можете задать любое название для любой вкладки. Подпись меняется сразу в панели без перезагрузки страницы. Если вы очищаете поле — скрипт возвращает оригинальную подпись поисковика.

### Перенаправить вкладку на другой сервис

Вместо стандартного раздела поисковика вкладка может открывать любой другой сайт. Например:

- Вкладка **Видео** в DuckDuckGo → открывает **YouTube** с тем же запросом
- Вкладка **Картинки** в Яндексе → открывает **Google Images**
- Вкладка **Карты** в Google → открывает **OpenStreetMap**

В поле URL можно использовать `{query}` — скрипт автоматически подставит текущий поисковый запрос.

### Добавить кастомную кнопку

Помимо стандартных вкладок, можно добавить любое количество собственных кнопок прямо в строку навигации поисковика. Кнопки поддерживают `{query}` и отображаются в том же стиле, что и родные вкладки.

Примеры использования:
- Кнопка «GitHub» → `https://github.com/search?q={query}`
- Кнопка «Amazon» → `https://www.amazon.com/s?k={query}`
- Кнопка «Translate» → `https://translate.google.com/?text={query}&sl=auto&tl=ru`


## Как пользоваться

1. Откройте любой поддерживаемый поисковик и введите запрос
2. В панели навигационных вкладок найдите кнопку **≡** (три горизонтальных черты) — она добавляется скриптом
3. Нажмите на неё — откроется боковая панель настроек
4. В секции **Стандартные вкладки** настройте каждую вкладку:
   - Включите тумблер **Скрыть**, чтобы убрать вкладку из панели
   - Измените **Подпись** — текст, отображаемый на кнопке
   - Выберите готовый сервис из выпадающего списка или вручную введите **URL редиректа**
5. В секции **Кастомные кнопки** нажмите **+ Добавить кнопку**, чтобы вставить свою ссылку
6. Изменения применяются мгновенно; закройте панель кнопкой **×** или кликом вне неё
7. Кнопка **Сбросить** возвращает все настройки к исходным значениям

> Если кнопка **≡** не появилась в строке вкладок, откройте настройки через меню Tampermonkey → скрипт → **⚙ Настроить вкладки поиска**.

## Шаблон {query}

В любом URL (редиректе или кастомной кнопке) можно использовать `{query}` — скрипт заменит его на текущий поисковый запрос, закодированный для URL.

**Примеры:**

```
https://www.youtube.com/results?search_query={query}
https://github.com/search?q={query}&type=repositories
https://www.amazon.com/s?k={query}
https://translate.google.com/?text={query}&sl=auto&tl=ru
```

При вводе запроса `смешные коты` ссылка `https://www.youtube.com/results?search_query={query}` превратится в `https://www.youtube.com/results?search_query=%D1%81%D0%BC%D0%B5%D1%88%D0%BD%D1%8B%D0%B5%20%D0%BA%D0%BE%D1%82%D1%8B`.

## Техническая информация

### Права доступа (Grants)

Скрипт запрашивает минимальный набор прав:

| Право | Зачем |
|---|---|
| `GM_getValue` | Чтение сохранённых настроек |
| `GM_setValue` | Сохранение настроек |
| `GM_addStyle` | Добавление стилей панели настроек |
| `GM_registerMenuCommand` | Запасная точка открытия настроек через меню Tampermonkey |

Скрипт **не отправляет никаких данных** во внешние сервисы.
