# CSST-tampermonkey-script
Custom Search System Tabs - tampermonkey script, that allows change deafult images, videos, buys and other things in your search system



Donwload tutorial:
1. Download [TamperMonkey](https://www.tampermonkey.net/)
2. Add downloaded script from repository to tampermonkey



Now it's only working with duckduckgo search system
